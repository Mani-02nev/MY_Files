summary: Python Basics 
1. Operators
     * Arithmetic operator : +,-,*,/,%,**
     * Comparison operator : ==,!=,<,>,<=,>=
     * Assignment operator : =,+=,-=,*=,/=,%=
     * Logical operator : and , or, not
     * Bitwise operator : &,|,^,<<,>>
     * Member ship operator : in , not in
     * Identity operator : is , is not
2. Date structures
     * List
          * Ordered,changeable,duplicates allowed
          * Ex: my_list=[1,2,3]
     * Tuple
          * Ordered,unchangeable list
          * Ex: my_tuple=(1,2,3)
     * set
          * Unordered,unique values
          * Ex : my_set={1,2,3}
     * Dictionary
          * Key values pairs
          * Ex : my_dic={"name":"mani","age":20}
     * String
          * Immutable values
          * Ex : s="Hello"
3. control Flow
     * if-elif-else
          * Make decisions
          * Ex : if age >= 18:
                print("adult")
     * For loop
          * Repeat over a range or list
     * while loop
          * Repeat with a condition
     * Loop controls
          * break:stop loop
          * continue:skip current step
          * pass:Do nothing
